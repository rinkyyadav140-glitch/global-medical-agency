# Global Medical Agency - Professional Website

A modern, responsive, and SEO-friendly website for Global Medical Agency, a pharmaceutical and medical products distributor in Azamgarh, Uttar Pradesh, India.

## Features

✓ **Responsive Design** - Mobile-friendly and optimized for all devices
✓ **Modern UI** - Clean, professional, and trustworthy appearance
✓ **Fast Loading** - Optimized performance and minimal dependencies
✓ **SEO Optimized** - Proper meta tags and semantic HTML structure
✓ **Accessibility** - WCAG compliant with proper heading hierarchy
✓ **No Dependencies** - Pure HTML, CSS, and JavaScript
✓ **Contact Form** - Working form with email integration
✓ **Google Maps** - Embedded location map
✓ **WhatsApp Integration** - Direct messaging capability
