# Global Medical Agency Website

## Description
This is a professional, responsive website for Global Medical Agency, a provider of pharmaceuticals and healthcare products. The site showcases product categories, company information, testimonials, and contact details. It's designed to be unique, attractive, and user-friendly, with features like dark mode, search functionality, and interactive elements.

## Features
- **Responsive Design**: Works on all devices (desktop, tablet, mobile).
- **Dark Mode Toggle**: Switch between light and dark themes.
- **Product Search**: Filter products by category.
- **Interactive Elements**: Toggle buttons for product details, animated counters, testimonial carousel.
- **Parallax Hero Section**: Dynamic background for an engaging homepage.
- **Contact Integration**: Clickable WhatsApp and phone links for easy communication.
- **Accessibility**: Proper alt texts, semantic HTML, and keyboard navigation.
